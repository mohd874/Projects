Forwarding demo.


In case you have javascript enabled and were not backtracking, you will be forwarded to http://www.sccoutway/web/forwarding/base.html. (That happens to be from where you probably came from).

If you came here by backtracking however, no forwarding happens. You could simply continue with back tracking.

View the source of this page to see how it works.  Please give me credit when you use my little hack. If forwarding should fail, that is probably where you came from. Sorry but this mechanism does not work with all browsers.
